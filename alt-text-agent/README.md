# alt-text-agent

Next.js app that finds images in your Contentful space missing alt text, uses a Contentful AI Action to generate alt text, and saves it as drafts.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and click **alt text me!** to run.

See [DEPLOY.md](DEPLOY.md) for deploying to Vercel and sharing with colleagues.
